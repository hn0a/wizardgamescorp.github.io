Informations importantes

Cette page regroupe les informations importantes concernant le site web de Wizard Games Corp et doit être utilisé par les évaluateurs de ce site web afin d'avoir des renseignements importants. Il contient des informations similaires à un fichier readme d'un projet informatique.

Introduction

Ce site web d'une entreprise spécialisée dans le jeu vidéo fictive est réalisé dans le cadre du module de programmation web du 4e semestre de la formation PGE de l'Efrei Paris. Toute ressemblance avec une entreprise existante serait purement fortuite et involontaire.

Auteurs

Les auteurs de ce site web sont Clément LAATAR, Jérémy CHEN et hana [hn0a], étudiants en école d'ingénieurs en informatique.

Sources des éléments utilisés

Les images utilisées sur ce site web proviennent toutes de sources libres de droits ou d'une licence Creative Commons nous autorisant à les utiliser. Vous trouverez ci-dessous le détail des sources des images utilisées sur ce site web :

Pour les images des jeux, elles sont toutes issues de projets Open Source créés sur Unreal Engine, et disponibles sur la marketplace Unreal Engine.

Pour les images des membres de notre équipe fictive, elles sont toutes issues du site web thispersondoesnotexist.com, un générateur d'images de personnes générées par intelligence artificielle. Les images sont donc fictives et ne représentent pas de personnes réelles, et sont libres de droits.

Pour les images annexes, comme le logo de l'entreprise ou les backgrounds de certaines pages, elles sont issues du site Pixabay une banque d'images libres de droits.

Pour les images du merch, elles ont été générées par nos soins via le site web de Streamlabs et leur outil de création de merch. Nous sommes donc autorisés à les utiliser tant que le logo présent dessus est libre de droit, ce qui est le cas.

Limitations du projet

Ce projet étant uniquement un projet scolaire, les boutons "Acheter" ne sont pas fonctionnels et ne redirigent pas vers une page de paiement. De même pour le panier, il n'y a pas de bouton acheter et le panier ne mène nulle part.

Informations supplémentaires sur le site web

Toutes les pages de ce site web sont codées en HTML et CSS, sans utilisation de framework CSS. Chaque page du site web est conforme à toutes les normes W3C.
